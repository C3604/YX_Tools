<br>
## 安装使用
<br>
** 1.整体下载之后解压出来放在D盘根目录 **

		GIT下载文件夹默认带有master后缀，自行重命名为YX_Tools再使用。

** 2.在您的计算机中添加环境变量 **

		ugii_vendor_dir=D:\YX_Tools	

** 3.开启您的NX,即可使用 **
